1. Map 接口(interface) -> Map提供了映射的关系，其中的元素是依照 鍵值對(key - value)的形式儲存，能夠實現根據key快速查找value -> Map中的鍵值對是依照 Entry 類型的對象實例形式存在 -> key 不可以重複，value值可以 -> Map 支持 泛型，EX: -> Map -> k 是 key 的泛型 -> v 是 Value 的泛型 2. HashMap 實現類 -> 是Map的實現類 -> HashMap 是 Entry對象 是無序排列 -> key值 和 value值 都可以為 null,並且 key 值不可重複 3. HashMap 實現類 -> 元素會依關鍵值由小至大排序（有序）
