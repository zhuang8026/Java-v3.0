1. Set(interface)接口及其實現類 --> HashSet -> Set 是元素無序並且不可以重複的集合，被稱為 集 -> HashSet 稱為 哈希集，是Set的一個重要的實現類 2. List 和 Set 差別： -> Set 是非重複資料的集合 (最後的資料) -> List 內部資料可以重複 -> 判斷物件存不存在集合內，Set 的複雜度為 o(1)，List為 o(N) -> 複雜度 -> o(1)表示基本語句的執行次數是一個常數，一般來說，只要演算法中不存在迴圈語句，時間複雜度就為o(1) -> 第一個for迴圈的時間複雜度為o（N），第二個for迴圈時間複雜度為o（N^2） 3. 補充： -> Set 繼承了 Collection -> Collection 繼承 Iterable -> Iterable 是 interface -> 所以 Set 是 interface -> HashSet() 是 Set 接口的 實現類
