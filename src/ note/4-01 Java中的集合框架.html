1. 集合 和 數組 非常相似，但數組的長度沒辦法自動擴展，集合卻可以。 -> 數組長度 固定 -> 集合長度 可變 2. 集合框架體系結構： 2-1 -> Collection （interface） -> List（序列）-> ArrayList(數組序列)(實現類) -> Queue（隊列）-> LinkedList(鏈表)(實現類) -> 存儲的數據是 排列有序的，並且可重複 -> Set（集）-> HashSet(實現類) -> 存儲的數據是 無序的，並且不可重複 2-2 -> Map（interface）-> HashMap(實現類) -> Map內部的數據結構是 < key, Value > 來顯示和存取（Entry 鍵值對）
