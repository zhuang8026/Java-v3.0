1. 什麼是 迭代？ -> 重复反馈过程的活动 -> 每一次对过程的重复被称为一次“迭代” 2. Iterator與ListIterator都是 interface ，皆可用來「走訪」或是刪除集合物件的元素 3. Iterator method -> Iterator iteractor() -> Iterator itr = list.iterator(); -> hasNext() => boolean -> next() => 回傳下一個元素 -> remove() => 刪除指定object 4. iterator() 接口 集合物件 -> Iterator itr = xxxx.iterator(); -> xxxx 是 List 物件 5. listIterator 接口 -> ListIterator itr = list.listIterator();
